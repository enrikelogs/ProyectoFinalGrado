# PoryectoFinalGrado
Reproductor de música en streaming.

## Descripción del Proyecto
Este proyecto fue mi Trabajo de Fin de Grado **(2019)**. Consiste en una **aplicación nativa para Android** capaz de gestionar y reproducir música mediante **streaming** desde un servidor remoto.

## Características Técnicas
Gestión de Datos: Integración con una base de datos relacional externa para la gestión de metadatos (canciones, álbumes y artistas).

Reproducción en Tiempo Real: Sistema de streaming con controles de flujo (Play/Pause, Skip, Back).

Sincronización de UI: Implementación de un SeekBar (slider) vinculado al progreso del audio en tiempo real.

Sistema de Búsqueda: Barra de navegación funcional para filtrar contenido por diferentes categorías.

Lógica de Reproducción: Algoritmo de reproducción aleatoria y gestión de colas.

## Estado Actual del Repositorio
[!IMPORTANT] Nota sobre el funcionamiento: Debido a que el proyecto data de 2019 y el alojamiento web donde residían la base de datos y los archivos multimedia ya no se encuentra activo, 
la aplicación no puede realizar el streaming actualmente. No obstante, el código fuente sirve como referencia de mi manejo de Java para Android, gestión de redes y arquitectura de aplicaciones.
